# Carpeta `files`

Esta carpeta sirve para centralizar archivos, documentos y assets que no forman parte del código fuente (ej.: PDFs, imágenes grandes, documentos de trabajo, descargas, etc.).

Sugerencias de uso:

- Mantén subcarpetas por tipo: `images/`, `pdfs/`, `docs/`, `assets/`.
- Añade un `.gitkeep` en subcarpetas vacías si quieres mantenerlas en el repositorio.
- Para archivos que no deben subirse al repositorio (ej. datos privados), añádelos al `.gitignore`.

Ejemplo de estructura:

```
/files
  /images
  /pdfs
  /docs
  README.md
  .gitkeep
```

Si quieres, puedo: crear las subcarpetas sugeridas (`images`, `pdfs`, `docs`) y actualizar el `.gitignore` del proyecto. ¿Lo hago?
